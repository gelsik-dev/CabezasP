# CabezasP (v1.4) – IIE Head Editor

> Editor rápido para generar y manipular **NBT de cabezas en Minecraft**.

---

## ¿Qué hace?

* Genera texto con rangos (`§`) + fecha
* Vista previa en tiempo real
* Convertidor en Base64 (codificación y decodificación)
* Formateador de `/give`
* Espacio libre para NBT (para hacer las cabezas más rápidamente)

---

## Ejemplo de lo que hace CabezasP

```txt
§8[§bJup§8] §cGelsikMC
§e§l/p h §6§lCabezas§b§lPandora
§7Actualizado el §n10/05/26
```

---

## Plantillas de CabezasP

```txt
{SkullOwner:{Id:"UUID",Properties:{textures:[{Value:"BASE64"}]}}}
```

```json
{"textures":{"SKIN":{"url":"URL"}}}
```
